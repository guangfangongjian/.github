# 光帆生态社区

我们是 [光帆科技](https://guangfan.com/) 的核心用户Github组织，旨在联动光帆官方、核心用户及开源社区力量，共建光帆生态。

## 组织成员
- [光帆共建](https://github.com/guangfangongjian) - 组织维护者
- [陪伴](https://github.com/peiban666)
- [山河尽赠佳人 / 奈何](https://github.com/naiheSH)

## 组织目标
- **生态共建**：联合光帆官方、用户和开发者，共同构建繁荣的光帆 AI 硬件生态
- **技术交流**：促进光帆相关技术的分享与讨论
- **开源协作**：通过开源项目推动光帆生态的创新与发展
- **用户服务**：为光帆用户提供更好的使用体验和技术支持

## 核心项目
- **ClawX**：从 [ValueCell-ai/ClawX](https://github.com/ValueCell-ai/ClawX) fork 并增强，集成了 [hi-light-plugin](https://github.com/Gongcong/hi-light-plugin)，为光帆用户提供更便捷的 OpenClaw 使用体验

## 如何参与
1. **关注我们**：Star 我们的仓库，了解最新动态
2. **加入讨论**：参与 issues 和 pull requests 的讨论
3. **贡献代码**：提交 bug 修复、功能增强或新特性
4. **分享经验**：分享你的光帆使用经验和技巧
